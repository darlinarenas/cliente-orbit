CAMBIOS IMPORTANTES - ORBIT ASSISTANT

1) Panel administrador > Productos:
Ahora el formulario permite cargar desde un solo lugar:
- nombre del producto
- SKU
- categoría
- imagen del producto
- estado del producto
- dificultad de instalación
- uso recomendado
- video instalación
- manual PDF
- descripción corta
- descripción larga
- ficha técnica: alcance, presión, uso, conexión y datos manuales
- guía de instalación con pasos
- productos recomendados para “Completa tu instalación”
- preguntas frecuentes

2) Cliente:
Después de escanear QR, el cliente ve solo la información cargada desde el administrador.
Si no llenas ficha técnica, guía o recomendados, aparecerá un aviso indicando que falta cargar esa información.

3) Base de datos:
El backend ahora guarda productos, QR, leads y escaneos en:
orbit-backend/data/orbit-db.json

Esto evita depender de localStorage para datos del proyecto.
Nota: sessionStorage se mantiene solo para guardar el token de sesión del administrador.

4) Subida:
Frontend:
git add .
git commit -m "actualizar orbit frontend producto admin"
git push

Backend:
cd orbit-backend
git add .
git commit -m "actualizar producto admin y db json"
git push
